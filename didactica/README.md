# Guia didactica del curso INF-220

Este directorio define un marco comun para que la parte practica del curso quede clara, medible y repetible en todas las unidades.

## Objetivo

Estandarizar como se enseña y evalua en INF-220:

- Secuencia didactica fija por clase y por unidad
- Ejercicios con criterios de evaluacion explicitos
- Practica por niveles A/B/C
- Rubrica unica para todo el semestre

## Archivos

- [01_plantilla_unidad.md](./01_plantilla_unidad.md): formato recomendado para redactar cada unidad
- [02_plantilla_ejercicio.md](./02_plantilla_ejercicio.md): formato de ejercicio evaluable
- [03_rubrica_unica.md](./03_rubrica_unica.md): rubrica transversal para laboratorios, tareas y proyectos
- [04_plan_practicas.md](./04_plan_practicas.md): plan de practica por niveles y por semana

## Uso recomendado

1. Definir objetivos medibles al inicio de la unidad.
2. Planificar una actividad Nivel A, una Nivel B y una Nivel C.
3. Publicar ejercicios usando la plantilla evaluable.
4. Calificar con la misma rubrica para mantener consistencia.
